Captions

Captions are more simple than pretty simple, actually, that was an exaggeration. But, I will still, pointlessly, give it a paragraph. So to make a caption, you use the captions tag, which looks like this '<caption> Caption Here! </caption>'. The reason the greater than/less than signs are curly, is just because they're unicode precedes and succeeds and you can't use actual greater than/less than signs inside a paragraph... (Yay, fillers!) Now, back to actual info, the only attribute you can give it is the align attribute. It looks something like this, 'align="left/right/top/bottom" '. You can use the align attribute in text to, it's why my headings have alignment. Okay, so next, I will go over the rowspan and colspan attributes on the <th> tag. I'm not completely sure if it works on the <td> tag. Note: The caption tag has to be used inside the table tag, but not inside a row.

Rowspan and Colspan

The rowspan and colspan are pretty self-explanatory. They make a <th> take up more than one row, or column, depending on which one you use. So if you did 'rowspan="3"', which is what the "CS Avatars" block uses, it will take up three rows. If you did 'colspan="4"', it would take up four columns, just as the "Meet Orange Juice Squid and Mr. Pink!" block uses. The attribute syntax the same as most; 'rowspan="rows" ', and the same applies to colspan, just, you know, with colspan instead... These two attributes are extremely useful in formatting, but I'm only sure they work with <th>. You would just use the tag inside a row, so far it works no problem. Then you would use <td> tag for everything else.

Borders

YAYY! Borders! Borders are AMAZZING... you can do so much with them, and not just in tables, the big fancy red border on the display is not part of the table, it's part of a div tag. Borders have eight different styles,

Groove Ridge Inset Outset Solid Double Dotted Dashed

Most people will probably use the 3-D ones when wrapping text like I did in the display, and above, and the 2-D ones for tables, or just 3-D... So, a border has three sub-attributes. The sub-attributes are (in order) the size (in px), the style, and the color. The CSS tag looks something like this,
'border: 6px, groove, green'. That is the code for the Groove display. You can do 'border-bottom' and it will only do it for the bottom, or 'border-top', any direction, and it will only do a border for that. Using that will give you different borders, but connected like the blue outside border in the display. Not only that, you can do 'border-style', 'border-width', or 'border-color', and you would only have to specify the width, or style. Or, if you're really specific... 'border-bottom-style', etc. The border attribute works on anything, really. But, you either have to do a seperate table for everything, so for the table heading blocks and table data blocks or on the table tag, just add border as an attribute, but don't assign it anything. (Or just the rows) The very last thing on borders, you can add the border collapse attribute and assign it collapse to collapse the borders. Oh dang, another last thing, there is also border spacing to add space in between borders. Mess around with it to get the hang of everything.
